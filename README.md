# DNC Lookup

**Do Not Contact (DNC)** lookup tool in Python using `pandas`, `numpy`, and `DuckDB`. It efficiently processes large datasets, cross-referencing contact lists to flag DNC entries. `DuckDB` enables fast SQL queries, while `pandas` and `numpy` handle data manipulation and numerical tasks, ensuring compliance in marketing campaigns

## Libraries Used:
1. [DuckDB](https://duckdb.org/)
2. [Pandas](https://pandas.pydata.org/)
3. [Numpy](https://numpy.org/)
